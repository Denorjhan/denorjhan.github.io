Projects

This Algorithm Visualizer consists of four sub projects, with each one demonstrating a unique set of algorithms or a different application in which they can be used in.

Path Finder

This program allows the user to see how different path finding algorithms determine their result by showing how the algorithm searches around obstacles in real time.

Number Guesser

The user will be asked to enter an upper limit and then think of a number between 0 and that upper limit. By using binary search, the program will then output the number the user is thinking of by asking simple yes/no questions.

Sorting Visualizer

This program shows how different sorting algorithm go about sorting a data set from smallest to largest. The user can also compare two different algorithms at the same time to see how they compare against each other.

Recursive Sorting Visualizer

Similar to the previous sorting visualizer, this program demonstrates sorting algorithms in action. However, this program only uses recursive algorithms so the user can see how each algorithm segments the data set.
